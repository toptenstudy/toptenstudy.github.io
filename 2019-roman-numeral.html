10. Roman numeral for 100,555


Answer:

Why is 555 in Roman Numerals Written as DLV? We know that in roman numerals, we write 5 as V, 50 as L, 500 as D, and 100 as C. Therefore, 555 in roman numerals is written as 555 = 500 + 50 + 5 = D + L + V = DLV.

(C)DLV

Explanation:

1. Break the number (decompose it).

Break the number (decompose it) into place value subgroups:

100,555 = 100,000 + 500 + 50 + 5;

2. Convert each subgroup.

Convert each subgroup to Roman numerals:

100,000 = (C);

500 = D;

50 = L;

5 = V;

3. Wrap up the Roman numeral.

Substitute the Roman numerals for each subgroup:

100,555 =

100,000 + 500 + 50 + 5 =

(C) + D + L + V =

(C)DLV

Roman numeral, any of the symbols used in a system of numerical notation based on the ancient Roman system. The symbols are I, V, X, L, C, D, and M, standing respectively for 1, 5, 10, 50, 100, 500, and 1,000 in the Hindu-Arabic numeral system.
