6. If we __________ harder we would have passed the UPCAT. a. had studied b. studied c. have studied d. study


The correct answer is a. had studied

So the complete sentence becomes:

If we had studied harder, we would have passed the UPCAT.

The sentence above is an example of using the conditional sentence type 3 pattern in English.

What is a conditional sentence type 3 or a third conditional?

Conditional sentence type 3 is the last type of conditional sentence type 1 2 3. This sentence has a function to be used in conditions where the result of the stated conditions will not be possible because the condition has been fulfilled in the past.

The pattern of the conditional sentence type 3 is :

If + Past Perfect + , + Subject + Would + Have + Past Participle

                                            or

Subject + Would + Have + Past Participle + If + Past Perfect

Example :

Tom would have passed the final exam if he had studied seriously.

Note : If he had studied seriously (condition), Tom would have passed the final exam (Result)

Explanation From the example above, it can be interpreted that the final exam has been held and Tom did not pass because he did not study seriously.
