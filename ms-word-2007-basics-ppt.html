4. create a simple essay about the importance of knowing the basic formatting options of MS word 2016 ​


Essay:

How important is it to know the basic formatting options of MS word 2016?

Microsoft Word is a powerful tool for creating documents, and one of the most important ways to use it is to know how to format your text. It's easy to get lost in the options that Word offers and forget about the basics, but you can get a lot done without spending too much time learning all of the fancy formatting options.

You might not know this, but there are three basic types of formatting: font, font size and color, and paragraph style. You can change each of these things individually or set them up with preset styles that give you more control over how your text looks.

For example, if you have a document with lots of bolding and italicizing throughout it, changing just one character in your text won't make much difference—but if you change all those characters at once by using bold or italics instead of just one character at a time? That'll make a huge difference!

The same goes for font size. If you want to make sure that everything on page 1-4 is big enough for people to notice without squinting their eyes or having their heads explode after reading it all at once? Sure thing! Just change the font from regular.

Essay #2:

MS Word 2016 is a powerful word processor, but it has a few formatting options that you should know about.

The first option is the format of your document. You can choose between two options: normal and simple. If you choose normal, your document will be formatted in the way it looks on paper. If you choose simple, however, your document will be formatted in the same way on all devices: print, computer screen, etc. The second option is how much space you want to allow for your document. You can choose between two options: standard or wide. For example if you want to make sure that there is enough space for all of the text in the document but not too much then you should use standard because wide would make the text too large and might not fit on some screens or printers (it depends on how many lines there are). Finally there is a third option called pagination which allows you to control how much space there is between pages in each chapter of your book (or whatever it may be).

11. What is Ribbon in Ms word 2007


Answer:

Microsoft introduced the Ribbon as a user interface feature in Microsoft Office 2007. It's next to the Quick Access Toolbar and below the Title Bar. Home, Insert, Page layout, References, Mailing, Review, and View are the seven tabs. Each tab contains a set of related instructions.

Answer:

The Ribbon is a user interface element which was introduced by Microsoft in Microsoft Office 2007. It is located below the Quick Access Toolbar and the Title Bar. It comprises seven tabs; Home, Insert, Page layout, References, Mailing, Review and View. Each tab has specific groups of related commands.
