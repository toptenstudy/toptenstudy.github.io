18. A GIRL PULLING HER CART?​


Answer:

It would depend if it would meet the requirements for determining work. In physics, we can say that work is done when a force is applied to an object, causing it to move. Another way to put this is that work is when distance achieved after a certain amount of force is applied. If force is applied but no distance has been achieved, then no work is done.

When is work done?

In this situation, it would depend if the cart she is pulling moved a distance, regardless of the amount of it. If distance has been achieved, then we can say that work is done. If she only pulled the cart but it did not move any distance, then no work is done.

In summary, the two factors that need to be present for us to be able to say that work is done is when:

Force is applied.

Distance has been achieved.

Examples

To help distinguish work done between no work done, we can study the following examples:

When you push a wall, there is force exerted but since the wall does not move a distance, then there is no work done

When you throw a ball, force is applied on the ball, and it flies off a distance. There is work done.

If someone lifts a box from the ground, then there is work done.

However, if someone only hold a box in position, then there is no work done. There is force applied here, though.

[tex]F_{net}[/tex] is the sum of all forces acting on an object. Opposing force is equivalent to [tex](-)[/tex]

[tex]m=10kg[/tex]

[tex]a=?[/tex]

Formula:

[tex]a=\frac{F_{net}}{m}[/tex] where [tex]a=acceleration[/tex]; [tex]m=mass[/tex];[tex]F_{net} =[/tex]net force
