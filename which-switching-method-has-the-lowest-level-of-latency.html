21. What is the main difference between Bandwidth and Latency?

29. There are two positive fractions. If we switch the denominators, the sum after the switch is 100 times the sum of the two positive fractions before the switch. Find the two fractions, expressed in lowest term.


Answer:

The two fractions are 1/1 and 199/201.

Step-by-step explanation:

Let the sum of the two fractions originally be [tex]\frac{1}{1}+\frac{a}{b}[/tex], it follows that the sum after the switch will be [tex]\frac{1}{b}+a[/tex].

From the given information, we set an equation

[tex]\displaystyle \frac{1}{b}+a=100\Big(1+\frac{a}{b}\Big)[/tex]

After multiplying both sides by [tex]b[/tex] and simplifying, we have

[tex]1+ab=100b+100a[/tex]

[tex]100b+100a-ab=1[/tex]

Note that

[tex]\begin{gathered} (a-100)(b-100)=ab-100a-100b+10000 \end{gathered}[/tex]

[tex]\begin{gathered} (a-100)(b-100)=-(100b+100a-ab)+10000 \end{gathered}[/tex]

Substituting [tex]100b+100a-ab=1[/tex] we have

[tex]\begin{gathered} (a-100)(b-100)=-1+10000=9999=99\times101 \end{gathered}[/tex]

This implies that

[tex]\begin{array}{c|c} a-100=99&b-100=101 \\ \boxed{a=199}& \boxed{b=201} \end{array}[/tex]

Therefore, the two fractions are 1/1 and 199/201.


30. what is the frequency and latency of RAM​


random axis memory

Contributor(s): Kevin Ferguson, Stacey Peterson and Rodney Brown

RAM (Random Access Memory) is the hardware in a computing device where the operating system (OS), application programs and data in current use are kept so they can be quickly reached by the device's processor. RAM is the main memory in a computer, and it is much faster to read from and write to than other kinds of storage, such as a hard disk drive (HDD), solid-state drive (SSD) or optical drive.

Random Access Memory is volatile. That means data is retained in RAM as long as the computer is on, but it is lost when the computer is turned off. When the computer is rebooted, the OS and other files are reloaded into RAM, usually from an HDD or SSD.
