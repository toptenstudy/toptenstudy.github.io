10. What are the Advance features of powerpoint?​


Answer:

PowerPoint offers word processing software, outlining, drawing, graphing, and presentation management tools- all designed to be easy to use and learn. These are some main & advance features of MS Powerpoint, Customize Color Schemes. Add Animation effects.

Remove the background from an image

The first PowerPoint tip that may be useful to know is how to remove the background from an image. Indeed, it is not uncommon to find an image that meets your need, but sometimes it will be better to isolate a single element to integrate it into your slide. What is more, it will also make it possible to have an image in the desired shape. This PowerPoint tip will therefore allow you to know how to do it. To do this, you will need to use the PowerPoint “Remove background” feature. You will therefore have to start by selecting the image from which you want to remove the background, then “Format image” and “Remove the background”.

Change the size of a PowerPoint file

Another PowerPoint tip that can be very useful is to know the PowerPoint feature that will reduce the size of your file. Indeed, certain presentations or PowerPoint models are sometimes too heavy to be sent by e-mail for example and it will therefore be necessary to be able to reduce the size of the file. Oftentimes, it is the embedded images that make the file large. This PowerPoint trick will therefore allow you to compress the images. By adding an image to your PowerPoint presentation, you will then have to click on “Compress images”.

Add a logo on his slides

Another PowerPoint tip to know is that you can add a logo in exactly the same place on all your slides. For this, it will be possible to create a PowerPoint template in which we will integrate its logo. But sometimes several models will be used for the same presentation, it is then on the Master Slide that it will be necessary to make the modifications to be applied to all the models, here the addition of a logo. To do this, you will have to click on the “Display” tab then “Slide master” to then make the desired modifications on the Master Slide.

Whereas the transition effects are limited to a single event per slide, animations can be applied to every object on a slide – including titles and other text boxes. Many objects can even have animation applied to different components, for example each shape in a SmartArt graphic, each paragraph in a text box and each column in a chart. Animations can be applied to three separate ‘events’ for each object:

Emphasis – an effect to focus attention on an object while it is visible

Exit – how the object disappears from the slide

To apply an animation effect, choose the object or objects to be animated, then choose Animation Styles or Add Animation from the Animations toolbar.
