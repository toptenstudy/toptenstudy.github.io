1. Groupware is also known as _____


Groupware

Groupware is also known as  collaborative software. Groupware is a software designed to facilitate collective working by a number of users. Its goal is to dispersed locations through shared Internet interfaces as a means to communicate within the group. Groupware also include remote access storage to archive used data files and can be accessed or retrieved.

28. A program or a set of programs used by people to achieve a common goal is called _____. Ans: Groupware


A program or a set of programs used by people to achieve a common goal is called GROUPWARE.

EXPLANATION:

Basically the understanding of Groupware or often referred to as Collaborative software is a device or application, computer software designed directly to be able to assist someone in doing various tasks together, of course the goal is also the same. The advantage of this software is that the work can be done simultaneously in different places, so of course it is very useful for the company. In the office usually each worker holds their own computer. But what they do is work together that has the same goal.

Of course, in this case, an office or company needs a groupware software to be able to collaborate in doing these joint tasks using their respective computers. Between these office employees can do every task, exchange information and even exchange files quickly, working on one file simultaneously in different situations can be done with the Groupware application. Peter and Trudy Johnson Lenz are the first experts to give the term to the notion of Collaborative Software, they mention that the definition is a group process that is done intentionally can be added using software to be able to support the work.
