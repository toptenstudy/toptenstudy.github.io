Don't forget to simplify your answer.

Additional Information:

Evaluating Functions

Finding the value of f(x) =... or y =... that corresponds to a given value of x is known as evaluating a function. Simply replace all of the x variables with whatever value has been provided to x. When we are asked to assess f(4), for example, we are given the number 4 for x.

Example: Given that f(x) = 2x + 2, find f(2)

Solution: We will evaluate the function once the value of x has been set to 2. The first step is to change each x to a 2. Then, using the order of operations, evaluate the function.

f(x) = 2x + 2

So, we have :

f(2) = 2(2) + 2

       = 4 + 2

       = 6
