10. The cost of two notebook and three folder is 705php. If the notebooks costs 40php more than the folder, find the cost of the folder and the notebooks.​


Answer:

The cost of two notebook and three folder is 705php. If the notebooks costs 40php more than the folder, find the cost of the folder and the nnotebook?

We can set up an equation to represent the problem using variables. Let x be the cost of one folder, and y be the cost of one notebook. Since the notebook costs 40php more than the folder, we know that y = x + 40.

We also know that the cost of two notebooks and three folders is 705php. So the total cost can be represented by the equation:

2y + 3x = 705

We can substitute the value of y from the first equation into the second equation:

2(x + 40) + 3x = 705

Simplifying this equation gives:

2x + 80 + 3x = 705

5x + 80 = 705

5x = 625

x = 125

So the cost of one folder is 125php. To find the cost of one notebook, we can substitute this value back into the first equation:

y = x + 40

y = 125 + 40

y = 165

So the cost of one notebook is 165php.

Therefore, the cost of two notebook and three folder is 705php. where cost of one notebook is 165php and cost of one folder is 125php.
