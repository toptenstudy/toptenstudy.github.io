5. Percent worksheet calculate the given percent of each value 92% of 3.9 =​


Answer:

the answer is 3.588

To get the solution, we are looking for, we need to point out what we know.

1. We assume, that the number 3.9 is 100% - because it's the output value of the task.

2. We assume, that x is the value we are looking for.

3. If 3.9 is 100%, so we can write it down as 3.9=100%.

4. We know, that x is 92% of the output value, so we can write it down as x=92%.

5. Now we have two simple equations:

1) 3.9=100%

2) x=92%

where left sides of both of them have the same units, and both right sides have the same units, so we can do something like that:

3.9/x=100%/92%

6. Now we just have to solve the simple equation, and we will get the solution we are looking for.

7. Solution for what is 92% of 3.9

3.9/x=100/92

(3.9/x)*x=(100/92)*x - we multiply both sides of the equation by x

3.9=1.0869565217391*x - we divide both sides of the equation by (1.0869565217391) to get x

3.9/1.0869565217391=x

3.588=x

x=3.588

now we have:

92% of 3.9=3.588

12. percentage base rate worksheet


Answer:

rate - usually have a percent sign (%) or is a rational number like a decimal.

base - usually is larger but can be known depending on the rate. If the rate is below 100%, the base is higher than the percentage and lower if otherwise.

percentage - can also be determined based on the rate.

In calculating for the base, rate, or percentage. We just need to use the formula...

[tex]percentage=base×rate \\ [/tex]

The formula above is used to find the percentage of a number.

Here are the formulas in getting the base and rate which is derived from the formula above.

[tex]\begin{lgathered}\begin{lgathered}base = \frac{percentage}{rate} \\ rate = \frac{percentage}{base}\end{lgathered}\end{lgathered} \\ [/tex]
