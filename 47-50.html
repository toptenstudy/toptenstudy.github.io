11. 47+48+49+50+...+127+128


Answer:

7175

Step-by-step explanation:

This is a two-step addition of consecutive numbers problem.

Whoever wrote this equation didn't expect us to be very diligent to add each number. You can easily solve this using a calculator, but you can also employ a formula to solve this.

The formula for the sum of 1+2+3+4+...+n is S = n(n+1)/2, where n is the final term, and S is the sum.

We need to use this formula two times.

To find the sum of the numbers 1 to 128.To find the sum of the numbers 1 to 46.

If we did that correctly, we can get the final answer by finding their difference.

Let's do Step 1 first.

S1 = (128)[(128)+1] / 2

S1 = (128)(129) / 2

S1 = 16512 / 2

S1 = 8256

Let's now do Step 2.

S2 = (46)[(46)+1] / 2

S2 = (46)(47) / 2

S2 = 2162 / 2

S2 = 1081

Now, we need to find the difference.

S = S1 - S2

S = 8256 - 1081

S = 7175

Therefore, the sum of the given addition expression is 7175.

Hope this helps!

Problem:

In a class of 50 students, the number of females is 2 more than 5 times the number of males. How many males and females are there in the class?

Solution:

Let x be the male students

Then, let 2 + 5x be the female students

There are 50 students in a class

Equation:

x + (2 + 5x) = 50

Solve the value of x.

x + (2 + 5x) = 506x + 2 = 506x = 50 - 26x = 48/6x = 8

Then, substitute the value of x to the female students.

2 + 5x2 + 5(8)2 + 4042

Answer:

Therefore, the females are 42 students and the male are 8 students.
