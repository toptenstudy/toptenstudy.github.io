From the first statement, we know that if Amy orders beef, Betty orders chicken. Therefore, if Betty ordered chicken today, Amy must have ordered beef yesterday.

From the second statement, we know that either Amy or Cheryl ordered beef, but not both. This means that if Amy ordered beef yesterday, then Cheryl must have ordered chicken yesterday. On the other hand, if Cheryl ordered beef yesterday, then Amy must have ordered chicken yesterday.

From the third statement, we know that Betty and Cheryl do not both order chicken. Therefore, if Betty ordered chicken today, then Cheryl must have ordered beef yesterday.

Putting all of this information together, we can conclude that Amy ordered beef yesterday and Cheryl ordered chicken today.
