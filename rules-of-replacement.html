Answer:

p ≡ ~~p. Double negation.

~(p • q) ≡ (~p ∨ ~q) ~(p ∨ q) ≡ (~p • ~q) ...

(p⊃q) ≡ (~p ∨ q) Material implication.

[p≡q] ≡ [(p⊃q) • (q⊃p)] [p≡q] ≡ [(p • q)∨(~p • ~q)] ...

(p ⊃ q) ≡ (~q ⊃ ~p) Transposition.

(p ∨ q) ≡ (q ∨ p) (p • q) ≡ (q • p) ...

[p ∨ (q∨r)] ≡ [(p∨q)∨ r] ...

[p • (q∨r)]≡[(p•q) ∨ (p•r)]

Evaluating Functions

Finding the value of f(x) =... or y =... that corresponds to a given value of x is known as evaluating a function. Simply replace all of the x variables with whatever value has been provided to x. When we are asked to assess f(4), for example, we are given the number 4 for x.

18. Use Rules of Replacement to show that the given propositions below are logically equivalent to each other. ¬ ( ∨ ) → ≡ ¬ ( ¬ → ) →


Answer:

Step-by-step explanation:

To show that p → q ≡ ¬ q → ¬ p using replacement rules, we need to show that they have the same truth value for all possible truth values of p and q.

Here's how we can do it using replacement rules:

p → q

≡ ¬ p ∨ q (replacement rule for conditional)

≡ q ∨ ¬ p (commutative law of disjunction)

≡ ¬ ¬ q ∨ ¬ p (double negation law)

≡ ¬ q → ¬ p (replacement rule for conditional)

Therefore, we have shown that p → q ≡ ¬ q → ¬ p using replacement rules.

25. PROBABILITY Hans and Ritz are playing a deck of cards. They want to apply the concepts of probability. Ritz said that solving the probability of drawing a spade and then a face card from a standard deck of card with replacement requires the multiplication rule while Hans claims that the addition rule must be applied. Whose argument is correct? Show your proof by indicating the complete and correct probability.


₊˚ ଘ Answer

Ritz's argument is correct.

Observing the problem above, we can see the clue word "and" which signifies the usage of an intersection of compound events. When solving for the intersection, the standard formulas are:

P(A) x P(B) ─────────> for independent eventsP(A) x P(B following A) ──> for dependent events

Ritz argued that multiplication is involved to solve for the probability of drawing a spade and then a face card from a standard deck of cards with replacement; he is correct.

₊˚ ଘ Solving

("with replacement" means it is independent)

P(AnB) = P(A) x P(B)

P(AnB) = (13/52) x (12/52)

P(AnB) = 3/52

The probability of drawing a spade and then a face card is 3/52 or 5.77%
