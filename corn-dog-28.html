3. The number of cans of corn sold varies directly with the number of packages of hot dog buns sold. If 23 cans of corn sell when 6 packages of hot dog buns sell, how many cans of corn are sold if 42 packages of hot dog buns are sold?


DIRECT VARIATION

[tex] \red{\overline{\qquad\qquad\qquad\qquad\qquad\qquad\qquad\qquad\qquad}}[/tex]

[tex]\small\bold {PROBLEM:}[/tex]

The number of cans of corn sold varies directly with the number of packages of hot dog buns sold. If 23 cans of corn sell when 6 packages of hot dog buns sell, how many cans of corn are sold if 42 packages of hot dog buns are sold?

[tex]\\[/tex]

Let c = number of cans of corn sold

h = number of packages of hotdog buns sold

[tex]\small\bold {MATHEMATICAL\: SENTENCE:}[/tex]

c varies directly as h

[tex]\small\bold {EQUATION:}[/tex]

c = kh[tex]\quad[/tex]where k is the constant of the variation

[tex]\small\bold {GIVEN:}[/tex]

c = 23 when h =6

[tex]\small\bold {UNKNOWN:}[/tex]

constant of the variation kc when h = 42

[tex]\small\bold {SOLUTION:}[/tex]

➨ First, we have to find the value of k using the given values.

[tex] \tt c = kh \\ \\ \tt 23 = k(6) \\ \\ \tt 6k =23\\ \\ \tt \frac{6k}{6} = \frac{23}{6} \\ \\ \boxed{ \tt k = \frac{23}{6} }[/tex]

➨ Now that we have the value of k, we can find c when h = 42

[tex] \tt c = kh \\ \\ \tt c = ( \frac{23}{6} )(42) \\ \\ \boxed{ \red{ \tt c = 161}}[/tex]

∴ There are 161 cans of corn if 42 packages of hotdog buns are sold.
