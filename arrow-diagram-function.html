1.) {(-2, -4), (-1, -3), (0, -2), (1, -1)}

The given x-coordinates are -2, -1, 0, and 1. Thus, the domain is [tex]\{-2, -1, 0, 1\}[/tex].The given y-coordinates are -4, -3, -2 and -1. Thus, the ranges are [tex]\{-4, -3, -2, -1\}[/tex].Since every value of x-coordinate have only one value of y-coordinates, it would be a function.The range on the set of ordered pair is two less than the domain. The equation would be [tex]y = x - 2[/tex]. Therefore, the function notation is [tex]f(x) = x - 2[/tex].
