1st entry: p(x) = 540

2nd entry: p(x) = 460

Step-by-step explanation:

Given:

p(x) = 640 - 0.2x

where x = number of printed T-shirts

Target number of shirts for 1st 3 entries:

1. 500

2. 900

3. 1300

First entry:

x = 500

We substitute x into the equation:

p(x) = 640 - 0.2x

p(x) = 640 - 0.2(500)

p(x) = 640 - 100

p(x) = 540

Second entry:

x = 900

We substitute x into the equation:

p(x) = 640 - 0.2x

p(x) = 640 - 0.2(900)

p(x) = 640 - 180

p(x) = 460

Third entry:

x = 1300

We substitute x into the equation:

p(x) = 640 - 0.2x

p(x) = 640 - 0.2(1300)
