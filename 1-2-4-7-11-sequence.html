22. what is the sequence pattern of 1 2 4 7 11 16 22 29​


The sequence pattern of 1, 2, 4, 7, 11, 16, 22, 29 is obtained by adding consecutive odd numbers starting from 1.

- We start with 1, then add the next odd number (1+1=2), which gives 2.

- Next, add the next odd number (2+2=4), which gives 4.

- Then, add the next odd number (4+3=7), which gives 7.

- Add the next odd number (7+4=11), which gives 11.

- Add the next odd number (11+5=16), which gives 16.

- Add the next odd number (16+6=22), which gives 22.

- And finally, add the next odd number (22+7=29), which is the last number of the sequence.

Therefore, the sequence is formed by adding consecutive odd numbers to the previous term.
