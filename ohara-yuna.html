7. Zeus can clean the classroom in 8 minutes. yuna can do the same in 11 minutes. if they work together how long will it take them to clean the classroom?


Answer:

4.63 minutes

Step-by-step explanation:

Work problems involve the same idea as speed problems where:

work = rate * time

rate = work * time  

time = work / rate

In this problem, we consider the work to be 1 classroom or simply 1.

In order to have a rate for each worker, we determine their rate per minute.

[tex]rate_{zeus} = 1/8 \\rate_{yuna} = 1/11[/tex]

Since they would be working together, we get their combined rate.

[tex]rate_{zeus+yuna} = 1/8 + 1/11 = 19/88[/tex]

With this, we can substitute it to the formula to get the time it takes for them to complete the task together.

[tex]time = 1/(19/88) = 88/19 = 4.631578947[/tex]

Therefore, it will take them 4.63 minutes to clean the room together.

If we were to write the whole process in a single equation it would be as follows:

[tex]time = 1/[(1 / time_{zeus}) + (1/time_{yuna})]\\time = 1/(rate_{zeus} + rate_{yuna})[/tex]
