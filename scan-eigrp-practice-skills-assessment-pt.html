1. Which eigrp parameter must match between all routers forming an eigrp adjacency?


Answer:

EIGRP routers must belong to the same autonomous system for an adjacency to be successful.


2. What is the purpose of the eigrp null0 summary route?


Answer:

The Null0 interface represents a route to nowhere. Using a route to the Null0 interface prevents routing loops for any destination networks present in the summary route that do not actually exist.


3. What is a disadvantage of using eigrp automatic summarization?


Answer:

It creates inconsistent routing when the network has discontiguous networks. When an organization has discontiguous networks, automatic summarization can cause inconsistent routing.
