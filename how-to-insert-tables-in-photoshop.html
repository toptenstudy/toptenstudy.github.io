16. how to insert a table using the main menu?


To create tables using the Insert Table dialog box:

Click Table on the menu bar.Select Insert and then Table from the cascading menu. The Insert Table dialog box appears.Determine the number of columns and rows you need in your table. You can add more later, but save yourself some work. You can always add rows by pressing Tab at the end of a row.To create a table as wide as your page, leave the Fixed Column Width setting on Auto.Click OK. A table is inserted into your document.

To create tables using the Insert Table button:

Click the Insert Table button.Drag the number of columns and rows you want in your table.

To create tables using the Draw Tables button:

Open the Tables and Borders toolbar by clicking View on the menu bar, then selecting Toolbars and Tables and Borders from the cascading menu. The Tables and Borders toolbar will appear.Click the Draw Tables button on the Tables and Borders toolbar. The mouse pointer turns into a pencil.Drag the pencil to create a rectangle about the size of the table you want.Release the mouse button. The border of the table appears in your document.Use the pencil again to draw column and row borders.Click the Draw Table button again to change the pencil back into an I-beam.

20. How to blend colors in photoshop


Answer:

There are two popular methods to blend colors in photoshop:

Color SwatchesColor Picker

It is easy to see these options when starting Photoshop because these are already displayed at default settings.

Explanation:

Color Swatches

By default, Color Swatches are colors prepared by default on Abode photoshop. You can pick several presets of colors with this or create your colors of personal choices. Of course, this will depend on the project you are working on.

Color Picker

This is accessed by double-clicking the Foreground and background color chooser seen at the toolbar (by default settings). This is the most common and efficient way of choosing colors as it shows a wide spectrum of the RGB colors. The Color Picker is commonly used by Digital Illustrators for concept arts, digital paintings, and illustrations because it is the easiest way to access color blending in Adobe PS.
