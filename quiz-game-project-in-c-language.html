Step-by-step explanation:

(x + 328) ÷ 5 = 85

x + 328 = 425

x = 425 - 328

x ≥ 97

Answer:

Step-by-step explanation:

The mean and the median are both measures of central tendency; they give an indication of the average value of a distribution of figures.

The median is the middle score in a list of scores; it is the point at which half the scores are above and half the scores are below. Medians are less sensitive to extreme scores and are probably a better indicator generally of where the middle of the class is achieving, especially for smaller sample sizes.

26. Create a simple 5-question quiz with three (3) choices that will display the score of the user. The quiz should be about your favorite game/series. Make use of control structures, arrays, and functions in your Java program


Answer:

import java.util.Scanner;

public class GenshinImpactQuiz {

   public static void main(String[] args) {

       Scanner scanner = new Scanner(System.in);

       int score = 0;

       String[] questions = {

           "1. What is the name of the main protagonist in Genshin Impact?",

           "2. How many elements are there in Genshin Impact?",

           "3. What is the name of the city located at the center of Teyvat?",

           "4. How many characters are in the starting roster of Genshin Impact?",

           "5. What is the name of the powerful being that has caused the destruction of many civilizations in Genshin Impact?"

       };

       String[][] choices = {

           {"A. Amber", "B. Barbara", "C. The Traveler"},

           {"A. 7", "B. 8", "C. 6"},

           {"A. Mondstadt", "B. Liyue", "C. Inazuma"},

           {"A. 12", "B. 20", "C. 16"},

           {"A. Barbatos", "B. Tartaglia", "C. The Abyss Order"}

       };

       String[] answers = {"C. The Traveler", "B. 8", "B. Liyue", "A. 12", "C. The Abyss Order"};

       for (int i = 0; i < questions.length; i++) {

           System.out.println(questions[i]);

           for (int j = 0; j < choices[i].length; j++) {

               System.out.println(choices[i][j]);

           }

           String userAnswer = scanner.nextLine();

           if (userAnswer.equals(answers[i])) {

               score++;

           }

       }

       System.out.println("Your score is: " + score + "/" + questions.length);

   }

}
