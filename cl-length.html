11. Arrange the following elements according to increasing atomic size; P,Al, Cl,Na,S. a.) Al,Cl,Na,P,S b.) Cl, S,P,Al,Na c,) Na,Al,P,S,Cl d.) S,Cl,P,Na,Al


The arrange of the elements below according to increasing atomic size : b.) Cl, S,P,Al,Na

Further explanation

The atomic radius shows the distance from the atomic nucleus to the electrons in the outer shell

From left to right in the period system, the atomic radius gets smaller, while from one group from top to bottom the atomic radius gets longer

The elements P, Al, Cl, Na, S. located in the same period, period 3 (has the same electron shell (3) but has an increasing number of electrons from left to right according to the atomic number)

The atomic number of each element is

Na = 11 Al = 13 P = 15 S = 16 Cl = 17

These atomic numbers show that Na has the smallest number of electrons (11) and Cl has the largest number of electrons (17)

Because from the left to right in the period system, the atomic radius gets smaller, then the size of the atom is also getting smaller, so the order of elements according to increasing atomic size:

b.) Cl, S, P, Al, Na

27. How many atoms of Cl (chlorine) are in 0.62 moles of Cl?


The number of atoms = 3.73 x 10²³

Further explanation

A mole is a unit of many particles (atoms, molecules, ions) where 1 mole is the number of particles contained in a substance that is the same amount as many atoms in 12 gr C-12

1 mole = 6.02.10²³ particles

While the number of moles can also be obtained by dividing the mass (in grams) by the molar mass of the element or molecule

Can be formulated :

N = n x No

N = number of gas particles

No = Avogadro number (6.02.10²³)

n = number of moles

 

Given

0.62 moles of Cl

Required

The number of atoms

Solution

Input the value :

[tex]\tt N=0.62\times 6.02\times 10^{23}\\\\N=\boxed{\bold{3.73\times 10^{23}}}[/tex]
