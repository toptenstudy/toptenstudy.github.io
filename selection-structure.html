30. Answer the following questions :What is the sequence control structure ?What is the purpose of the selection structure ?How can the repetition control structure be used ?​


1. Control flow structure refers to the predetermined way in which a program's instructions are executed sequentially in the order in which they are written in the code.

2. The purpose of the choice structure is to allow the program to make decisions and take different actions depending on whether certain conditions are true or false. This allows for more complex and flexible behavior from the program since it can react differently to different situations.

3. The repeat control structure, also known as a loop, allows the program to repeat a set of statements multiple times, either for a fixed number of iterations or until a specified condition is met. This is useful for tasks that require the same operations to be performed on multiple inputs or for tasks that need to be repeated a specified number of times until a certain condition is met. There are several types of loops, including the while loop, the do-while loop, and the for loop, each with their own specific uses and exit conditions.
