Answer:

Xander ordered ₱105.50

Step-by-step explanation:

₱758.50 - ₱20 = ₱738.50

Xander orders ₱X amount.

Xyza orders 3*₱X

Xyril orders 3*₱X + ₱20.00, 1 + 3 + 3 = 7 orders.

₱738.5 / 7 orders = ₱105.50 is Xander’s order.

₱105.50 x 3 = ₱316.50 is Xyza’s order.

₱316.50 + ₱20 = ₱336.50 is Xyril’s order.

Check: ₱105.50 + ₱316.50 + ₱336.50 = ₱105.50 + ₱653 = ₱758.50
