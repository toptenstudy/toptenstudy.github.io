What is Software Licensing?  

The software license refers to the ways and terms which the defined software can be used in a legal way. Such way includes how the software can be distributed, copied and transferred. There are also solutions enumerated in any case that the software doesn't performed it's intended function.  

What is Software User Guide?  

The software user guide is a step-by-step procedure and guide on how to set-up the software, it's requirements, it's installation, and how to use it. The user guide also contains basic troubleshooting tips like what the user can do when they encountered some problems.

Explanation:

Explanation:

User documentation helps ensure your customers or users actually learn how to get the most out of your product. While a software license, establishes the rights of all parties involved with the software: the author, the provider and the end users.

10. a combination of operating system utility programs and applications are essential to the computer which of the following is being described based on the statementA. Application softwareB.Software packagesC. Security softwareD.System software​


Answer:

D. System Software

Explanation:

System software is required in order for a PC to boot Windows, MacOS, Linux, etc.

Reasoning:

Why not A? Application software is a software that accomplishes certain tasks for the user. Examples of application software is Photoshop, MS Word, etc. It is not related to a computer's ability to boot and function.

Why not B? Software packages is a bundle of software designed to work together. One example of this is Microsoft Office consisting of MS Word, MS Powerpoint, MS Excel, etc. It is completely not related to a computer's essentials in order to boot.

Why not C? Security Software is a software that is used in order to protect the data that can be accessed by users with differing levels of authority. This can also protect from malicious attacks i.e viruses. It does not have any effects on booting up a computer.

26. the nayon homeowners association generated Php,26,000.00 from the sale of tickets for a fund raising show.Tickets for adult cost Php.150.00 while tickest for children cost Php,100.00.If 1.12 tickets for adults were sold,how many tickets for children were sold?


Answer:

Tickets are counted in WHOLE NUMBERS. Based on the given, the number of tickets are in decimal form and NOT in whole numbers. Therefore, the given information for the problem is invalid.

Anyway, here's a step by step process for the same kind of problem:

Total cost of all tickets sold (T) = ₱26,000

Adult ticket's price (A) = ₱150

Child ticket's price (C) = ₱100

No. of adult tickets sold (a) = 1.12

No. of child tickets sold (c) = ?

Derivation of Formula:

T= Aa + Cc

T - Aa = Cc

(T - Aa)/C = c

c = (T - Aa)/C

Substitution of Given:

c = (26,000-(150)(1.12))/100

c = 258.32 or 259 (to make it whole)

Therefore, 258.32 tickets for children were sold.
