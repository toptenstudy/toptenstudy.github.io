12. The management of a local grocery store wants to estimate the average amount their customers spend at the store to within 0.50 pesos, with a 90% confidence. What is the minimum sample size required, if the standard deviation is assumed to be 4.50 pesos.


In order to solve this problem, we first have to list all the given values and identify the required value.

Given: margin of error = 0.50 pesos
confidence level = 90%
standard deviation = 4.50 pesos
Required: sample size

The general formula for calculating margin of error is,

[tex]margin \: of \: error = z \times \frac{s}{ \sqrt{n} } [/tex]

where s is standard deviation and n is sample size.

Since the confidence level is already given, we can easily look for the value of z in the z table. The z value for this problem is 1.645.

Next, we have to transform the equation into our working equation. Thus, looking for n we have the transformed equation:

[tex]n = ({ \frac{z \times s}{margin \: of \: error} })^{2} [/tex]

Substituting the given values, we have:

[tex]n = ({ \frac{1.645 \times 4.50}{0.50} })^{2} [/tex]
n = 219.19
or rounded up to 220

The minimum sample size then is 220.
