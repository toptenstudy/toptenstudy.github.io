Answer:

Internet-based program and web based program are two different types of programs. The difference between them is that internet-based program is a software that can be accessed from the internet whereas web based program is a software that you can access from your computer or mobile phone.

Internet-based programs are more user friendly than web based programs because they allow users to access the programs from any location at any time even when they are not in front of their computers. Web based programs, on the other hand, require users to use their computers or mobile phones to access the program which makes it difficult for them to access the program during off hours like weekends and holidays.
