4. Fundamental concepts of accounting


Record keeping, accounting's main purpose, is one of the subject's fundamental aspects. When storing and keeping information, a corporation must follow industry standards to ensure that it can be retrieved when required. Records must be stored completely and accurately in order to serve all transaction-related functions.

All the paperwork and books used to create financial statements or records necessary for audits and financial reviews are referred to as accounting records. Accounting records are any supporting papers, such as checks and invoices, as well as records of assets and liabilities, financial transactions, ledgers, and journals.

There are typically two types of accounting records: single entry and double entry. The one entry approach, as implied by its name, is significantly easier to use and performs better for minor activities. The more complicated double entry approach necessitates two entries, one credit and one debit, for each transaction a business conducts. To achieve bookkeeping balance and track the flow of money inside a company. Larger firms tend to do this more frequently, which aids in identifying mistakes and possible fraud.

18. Find the fundamentals identity of[tex] \frac{tan \: x \: + \: cot \: x}{cot \: x} [/tex]​


SOLUTION

In order to find the Fundamental Identity given trigometric expression, first we need to express it with [tex] cos [/tex] and [tex] sin [/tex]. Then, we use Pythagorean and Trigometric Identity.

[tex] \rm{\frac{tan \: x \: + \: cot \: x}{cot \: x}} \\\\ \rm{= \frac{\frac{cos \ x}{sin \ x} + \frac{sin \ x}{cos \ x}}{\frac{cos \ x}{sin \ x}}} \\\\ \rm{= \frac{\left(\frac{cos \ x}{sin \ x}+\frac{sin \ x}{cos \ x}\right)sin \ x}{cos \ x}} [/tex]

Join the two fractions in numerator

[tex] \rm{= \frac{\frac{cos^2 \ x + sin^2 \ x}{sin \ x \ cos \ x} sin \ x}{cos \ x}} \\\\ \rm{= \frac{cos^2 \ x + sin^2 \ x}{cos \ x \ cos \ x}} \\\\ \rm{= \frac{cos^2 \ x + sin^2 \ x}{cos^2 \ x}} \\\\ \rm{= \frac{1}{cos^2 \ x}} \\\\ \boxed{\rm{= sec^2 \ x}} [/tex]
