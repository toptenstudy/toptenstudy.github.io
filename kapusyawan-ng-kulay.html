3. 5 halimbawa ng kulay kapusyawan,kadiliman,katingkaran at malamlam?


              Ang mga kulay (Ingles:colour;Kastila:color) ay mga katangiang bahid ng mga bagay na nakikita ng mga mata ng tao na maaaring matingkad o mapusyaw. Nanggagaling ang kulay ng kalikasan mula sa sinag ng araw. Pinaghihiwahiwalay ng bahaghari ang lahat ng mga sinag ng araw. Makikita natin sa loob ng bahaghari ang kadalisayan ng bawat kulay. Ang kulay ay may mga katangian ito ay ang Values at Intensities.  Ang  values kapusyawan  o  lightness at kadiliman o Darkness. Halimbawa, ang kapusyawan ng pula (red)  ay rosas (pink) at ang kadiliman naman nito ay maroon. Ang intensity naman ng kulay ay nangangahulugang katingkaran o brightness at kalamlaman o dullness ng kulay. Halimbawa ng intensity ay, ang isang kulay ay dinagdagan ng puti o itim.

7. sangkap ng kulay na tumutukoy sa kapusyawan at kadiliman nitoA. espasyoB. foregroundC. MaranaoD. overlapE. value​


Answer:

E. Value

Explanation:

Value: Sangkap ng Kulay na Tumutukoy sa Kapusyawan at Kadiliman Nito

Ang sangkap ng kulay na tumutukoy sa kapusyawan at kadiliman nito ay tinatawag na value. Inilalarawan ng value ang bawat kulay sa kung gaano ito kalapit sa puti o sa itim. Kapag mababa ang value ng isang kulay, may kadiliman ito. Samantala, mataas naman ang value ng isang kulay kapag ito'y may kapusyawan.

Sa lahat ng mga kulay, ang puti ang may pinakamataas na value habang ang itim ang may pinakamababang value. Kaya naman, walang ibang kulay ang mas mataas sa value ng puti at wala ring ibang kulay ang mas mababa pa sa value ng itim.

Kahalagahan ng Value na Sangkap ng Kulay sa Larangan ng Sining

Ang bawat isang kulay ay puwedeng magkaroon ng higit sa isang value. Mayroong mapusyaw, katamtaman, at madilim na uri ng isang partikular na kulay depende sa value nito.Dahil sa value, nagkakaroon ng iba't ibang pagtingin sa isang kulay at maaari ring makalikha ng mga tatlong-dimensyonal na larawan (3D) gamit ang mga ito. Kung iisa lamang ang value ng bawat kulay, magmimistulang patag ang lahat ng mga bagay na nakikita ng mata. Ang pagkakaroon ng value ng isang kulay ang gumagabay sa ating mga mata upang makita ang pagkakasalungat ng mga kulay at anyo ng mga bagay.
