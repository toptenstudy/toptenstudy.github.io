25. find the numbers whose sum is 240, if the product of the square of one by the cube of the other is to be maximum.


Solve for minima and maxima

Let x = 1st number

y = 2nd number


Equation 1:

x + y = 240

x = 240 - y


Equation 2:

n = xy³

Substitute x from Equation 2 using Equation 1

n = (240 - y)y³

n = 240y³ - y⁴


Use derivative

d/dy (n) = 240y³ - y⁴

dn/dy = 3(240y²) - 4y³ = 0

720y² - 4y³ = 0

Factor

4y² (180 - y) = 0

Use 180 - y = 0

y = 180


Substitute y

x = 240 - 180

x = 60

Answer: The 1st number, the minima is 60 and 2nd, the maxima is 180.
