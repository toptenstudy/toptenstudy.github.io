12. Q_The table shows the nutritional combination of a food product. Construct the pie chart using the following tableNutrition carbohydrates proteins Vitamin and minerals Fat Mass in gram 120 135 15 30


To create a pie chart, we need to find the percentage of each component in the total mass of the food product.

First, we can calculate the total mass of the food product by adding up the mass of all the components:

Total mass = 120g + 135g + 15g + 30g = 300g

Next, we can find the percentage of each component by dividing its mass by the total mass and multiplying by 100:

Carbohydrates: (120g / 300g) x 100% = 40%

Proteins: (135g / 300g) x 100% = 45%

Vitamins and minerals: (15g / 300g) x 100% = 5%

Fat: (30g / 300g) x 100% = 10%

We can now use these percentages to create a pie chart. We can divide the circle into four sectors corresponding to the four components, and label each sector with the component and its percentage.
