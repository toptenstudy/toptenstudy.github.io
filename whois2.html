1. what are the two numbers whose sum is 1 and and whoose product is -12​


Answer:

4 and -3

Step-by-step explanation:

Let,

x = first number

y = second number

Solution:

x + y = 1

y = 1 -x ; Eq. 1

x • y = -12 ; Substitute y with Eq. 1

x • (1-x) = -12

x - x² = -12

[-x² + x + 12 = 0] (-1)

x² - x - 12 = 0

(x - 4) (x + 3) = 0

x - 4 = 0 x + 3 = 0

x = 4 x = -3

Using Eq. 1, substitute x with 4

y = 1 - x

= 1 - 4

y = -3

Using Eq. 1, substitute x with -3

y = 1 - x

= 1 - (-3)

= 1 + 3

y = 4

First pair: x=4 ; y=-3

Second pair: x=-3 ; y=4

Thus, the two numbers are 4 and -3.
