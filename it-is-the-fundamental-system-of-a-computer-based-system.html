3. What is the fundamental system of a computer based system?​


A computer system is broken down into three distinct units in order to perform its operations. They are the following: 1) Control unit, 2) Central processing unit, and 3) Arithmetic logical unit. The term "functional unit" refers to all three of these units.  

1 Logical Arithmetic Unit (ALU)

The Arithmetic Logical Unit handles the processing of the data and commands. The ALU primarily performs addition, subtraction, multiplication, division, logic, and comparison operations.

2. Control Unit (CU)

The Control Unit is the next part of the computer, acting as a supervisor to ensure that tasks are completed correctly. The execution order of computer programs and instructions is decided by the control unit.

3. Central Processing Unit (CPU)

The central processing unit (CPU) of a computer system consists of both the ALU and the CU. The CPU is often referred to as the computer system's brain.

16. composed of at least one operating system an a number of fundamental system utilities​


Answer:

This statement describes a typical computer system, which is composed of at least one operating system and a number of fundamental system utilities.

An operating system (OS) is a software program that manages the hardware and software resources of a computer. It provides an interface for users to interact with the computer and also provides a platform for other software programs to run on. Some examples of operating systems include Windows, MacOS, and Linux.

System utilities are software programs that help maintain and optimize the performance of the computer. These programs perform a variety of tasks, such as managing files and directories, managing memory and processing resources, and monitoring and troubleshooting system performance. Some examples of system utilities include disk defragmenters, backup programs, and system monitoring tools.

Together, the operating system and system utilities form the foundation of a computer system, providing the necessary tools to manage and maintain the system, and to run other software programs.
