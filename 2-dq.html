19. [tex]6t^{3} - 5q^{2} + 7t^{2} - q + 7 = 0[/tex]find dt/dq using implicit differentiation ​


Answer:

To find dt/dq using implicit differentiation, we need to differentiate both sides of the equation with respect to q.

Starting with the given equation:

6t³ - 5q² + 7t² - q + 7 = 0

Differentiating both sides with respect to q:

(18t²) dt/dq - (10q) + (14t) dt/dq - 1 = 0

Grouping the terms with dt/dq:

(18t² + 14t) dt/dq = 10q + 1

Dividing both sides by (18t² + 14t):

dt/dq = (10q + 1) / (18t² + 14t)

Therefore, the expression for dt/dq using implicit differentiation is (10q + 1) / (18t² + 14t).

27. D. PQRS is an isosceles trapezoid with median MD.13. If SM = 4x+5 and MP = 7x-10, what is RD? DO14. If RD=8-2x and DQ=2x+2, what is SM? MP?​


Answer:

13. RD = 25, DQ = 25

14. SM = 5, MP = 5

Step-by-step explanation:

I tried drawing the isosceles trapezoid and I came up with the image attached.

An isosceles trapezoid is a trapezoid whose non-parallel sides are congruent. In the diagram, SP = RQ.

The median of a trapezoid connects the midpoint of the non-parallel sides. In the diagram, M is the midpoint of SP, and D is the midpoint of QR. Via the definition of a midpoint SM = MP, and QD = DR.

Since SM = MP, we can substitute the values given and equate them

[tex]SM = MP\\4x+5 = 7x-10\\5+10 = 7x-4x\\15 = 3x\\5 = x\\x = 5[/tex]

Substituting x back to the equation gives us:

[tex]4x + 5 = \\=4(5) + 5\\=25[/tex]

SM = 25

MP = 25

To find out what RD is we check back with the earlier definitions we established.

SP = RQ because the trapezoid is isosceles.

Since MD is a median:

M is the midpoint of SP; D is the midpoint of RQ.

SM = MP and RD = DQ.

Via segment addition postulate,

[tex]SM + MP = SP[/tex], and [tex]RD + DQ = RQ[/tex]

Since SP = RQ, we can equate both of them

[tex]SM + MP = RD + DQ[/tex]

Since RD = DQ (midpoint), we can substitute one for the other.

[tex]SM + MP = RD + RD\\SM + MP = 2RD[/tex]

since SM = MP, we can substitute one for the other.

[tex]SM + MP = 2RD\\SM + SM = 2RD\\2SM = 2RD\\SM = RD\\[/tex]

So we conclude that SM = RD. Via transitive property, SM = MP = RD = RQ. All of the segments are congruent to each other.

Since SM = 25, RD is also 25, DQ is also 25.

For the second item, we shall do the same. Since D is the midpoint of RQ. RD = DQ. We can equate their values.

[tex]RD = DQ8-2x = 2x+28 - 2 = 2x+2x\\6 = 4x\\x = 6/4\\x = 3/2[/tex]

Substituting back to the original equation gives us

[tex]8-2x = \\8 - 2(3/2)\\8 - 3\\5[/tex]

So [tex]RD = DQ = 5[/tex]

Earlier, we proved that all four of the segments, RD, RQ, SM and MP are congruent to each other. So SM = 5, MP = 5.
