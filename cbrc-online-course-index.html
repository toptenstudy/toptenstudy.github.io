5. when there is no indicated index in a radical like √a it is implied that the index is ?


Answer:

The n in root(n,a) (always a natural number greater than 1) is called the index or the order of the radical, and a is called the radicand. When there is no indicated index, as in root(a), the index 2 is implied and it is read “the square root of a." When the index is 3 as in root(3,a), it is read “the cube root of a."

When a radical has no index, the square root is understood to be the one desired. For example, indicates the square root of 36. The line above the number whose root is to be found is a symbol of grouping called the vinculum.

26. What is High Index Diversity?What is Low Index Diversity?


A diversity index is a quantitative measure that measures how many different types there are in a dataset, and at the same time takes into account how equally the basic entities are allocated/distributed among those types.

A High Index Diversity means when the richness and evenness increases so does the diversity.

A good example of that is when a database is dominated by one or two types only, it is considered less diverse than one that have more types that have similar allocation of entities.

29. ano ang price index at konsyumer price index


Answer:

Ang Price Index ay isang panukat ng pagbabago ng presyo ng isang partikular na basket ng mga kalakal at serbisyo sa loob ng isang takdang panahon. Ito ay nagpapakita kung paano nagbabago ang kabuuang halaga ng mga kalakal at serbisyo na kasama sa basket sa paglipas ng panahon.

Sa kabilang banda, ang Consumer Price Index (CPI) ay isang uri ng price index na sumusukat sa pagbabago ng presyo ng mga bilihin at serbisyo na karaniwang binibili ng isang tao sa pang-araw-araw na buhay. Ito ay tumutulong sa pagsukat ng inflation rate at pagbabago ng halaga ng pera sa loob ng isang takdang panahon. Sa madaling salita, ito ay tumutukoy sa pagtaas o pagbaba ng mga presyo ng mga bilihin at serbisyo na karaniwang binibili ng mga mamimili.
