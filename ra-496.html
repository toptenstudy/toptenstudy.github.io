26. f(x)=2x²-x ,what is the value of x if f(x)=496


To find the value of x if f(x) = 496, we can substitute the value of f(x) into the equation and solve for x.

So we have:

f(x) = 2x² - x

496 = 2x² - x

Rearranging the terms, we get:

2x² - x - 496 = 0

Now, we can solve for x using the quadratic formula:

x = (-b ± sqrt(b² - 4ac)) / 2a

where a = 2, b = -1, and c = -496.

Plugging in these values, we get:

x = (-(-1) ± sqrt((-1)² - 4(2)(-496))) / 2(2)

x = (1 ± sqrt(1985)) / 4

So the two possible values of x are:

x = (1 + sqrt(1985)) / 4

x = (1 - sqrt(1985)) / 4

Both values satisfy the equation f(x) = 496.
